## Type declaration

### layers

> **layers**: (`null` \| `object`)[]

The layers matching the given IDs as an array. Any layers that don't
exist will be represented by a `null` in the array.

See [Layer](../../client/type-aliases/Layer.md) for the structure of the layer object.
